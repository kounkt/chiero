2. アクセス統計について

当サイトは、どのページがどれくらい見られているかを把握するために、Cloudflare, Inc. が提供する Cloudflare Web Analytics を利用しています。これは Cookieを発行せず、閲覧者を個人として特定しないプライバシー配慮型の統計です。取得するのは、ページの表示回数、参照元（どのサイトから来たか）、おおまかな国、画面サイズといった、個人を識別しない集計情報のみです。

一方で、当サイトでは以下を一切行っていません。

  • Cookieの発行
  • 広告の配信、および広告目的のトラッキング
  • 閲覧者個人を特定・追跡すること
  • フォームによる個人情報の入力・送信（お問い合わせは公式LINEのみ）

3. 外部サービスへの通信について

当サイトには、以下の外部サービスから配信される画像が含まれています。これらのページを表示すると、各社のサーバーにあなたのIPアドレスやブラウザの情報が送信されます。これは当社が取得するものではなく、各社の管理下に置かれます。

  • note（記録セクションの画像 / assets.st-note.com）— noteのプライバシーポリシー

また、当サイトは GitHub Pages（GitHub, Inc.）でホスティングしており、同社がアクセスログを保持する場合があります。アクセス統計（第2条）についても、集計は Cloudflare, Inc. のサーバー上で行われます。同社の取り扱いについては同社のプライバシーポリシーをご確認ください。
