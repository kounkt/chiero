# CHIERO公式サイト 戦略設計書(2026-06-12)

## サイトの役割
コーポレートサイトとLPの統合 = 「旗艦」。SNS(X/IG/TikTok/note)から来た見込み客を受け止め、
信頼を形成し、深さ別の導線(無料→書籍→コーチング)に振り分ける。CVポイントは公式LINE(lin.ee/YCCoRsBu)。

## マーケット
- 日本のコーチング市場: 2015年約50億→2022年約300億円と急成長。米国の1/50で伸びしろ大
- AI時代の二極化: 安価な伴走はAIへ / 経営の意思決定に踏み込む高付加価値対話は人間へ
- エグゼクティブコーチング相場: セッション5〜15万円、半年契約60〜100万円超
- → ポジション: 「AIを知り尽くした人間(10億バズ)が、AIにできない認知の仕事をする」

## ターゲット(優先順)
1. 経営者・個人事業主(年商数千万〜数十億): 売上はあるが次のフェーズで詰まり。最大インサイトは
   「経営者の孤独」(重要課題を相談できる相手がいない=小規模事業者の46%)

## コアメッセージ
- H1: 「成果はあるのに、自由ではない。」(顧客の痛み起点。診断LPと統一)
- サービス定義: 「占いのようにわかりやすく。コーチングのように深く。コンサルのように現実が動く。」
- メソッド: 言語化する → 抽象度を上げる → 現実を動かす
- 旧ヒーロー「頑張らず、本来の自分で、自由に。」は世界観コピーとしてブランド側に残置可

## コンバージョン設計
- 主CTA: 公式LINEで無料相談(ヘッダー・ヒーロー・プログラム・帽子の4箇所)
- 副CTA: 自由タイプ診断(/shindan/ サイト内ホスティング、無料・1分・登録不要 → 結果画面からLINE)
- 信頼形成: 権威ストリップ(著書200冊超/フォロワー40万/10億バズ/王様のブランチ) → 痛みの言語化(5項目)
  → 声6件(属性つきBefore/After+免責) → 成果数値(410万/400万超/50万フォロワー増/年商2桁億)
- 不安除去: FAQ5問、「無理な勧誘はしません」明記、枠限定(月数名=事実ベースの希少性)
- 価格: 非公開・無料相談で案内(Mindset型。高単価帯の標準)
